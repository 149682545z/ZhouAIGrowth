# 通用对话工具 (General Chatbots)

除了写代码，我们也需要处理文本、生成文档、进行逻辑推演。以下是目前的主流选择。

## 1. ChatGPT (OpenAI)
目前的行业标杆，能力最全面。
- **型号**：GPT-4o (Omni) 是当前主力，支持多模态（文本、图像、语音实时交互）。
- **擅长**：
    - 复杂的逻辑推理。
    - 创意写作、故事生成。
    - 广泛的通识知识问答。
- **注意**：免费版有使用限制，Plus 版体验最佳。

## 2. Claude (Anthropic)
以强大的长文本处理能和更像人的语调著称。
- **型号**：Claude 3.5 Sonnet 是目前的旗舰，代码能力极强。
- **擅长**：
    - **阅读长文档**：支持超长上下文（200k+ tokens），可以扔进去几十页 PDF 或整个代码库的压缩包。
    - **代码编写**：生成的代码往往比 GPT-4 更简洁、更能一次跑通。
    - **Artifacts**：可以直接预览生成的 React 组件、Mermaid 图表，非常适合前端原型设计。

## 3. DeepSeek (深度求索)
国产大模型的佼佼者，性价比和推理能力令人印象深刻。
- **型号**：DeepSeek-V3 (通用) 和 DeepSeek-R1 (推理强化)。
- **擅长**：
    - **中文理解**：对中文语境、成语、网络梗的理解更好。
    - **数学与代码**：R1 版本在数学推导和代码生成上表现优异。
    - **成本**：API 价格极低，适合大规模部署或个人开发者低成本实验。

## 4. Midjourney
图像生成的王者。
- **用途**：生成 UI 素材、配图、甚至 Logo 草图。
- **使用方式**：目前主要通过 Discord 交互。
